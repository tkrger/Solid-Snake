⬇    -   lefelé<br />

# Követelmény

Játék futtatásához Python és Pygame szükséges.<br /> 
Pygame telepítése előtt érdemes megnézni, hogy telepítve van-e a Python.<br /> <br />

## Python ellenőrzése
Parancssorban futtatva ellenőrizhetjük a jelenleg telepített Python verzióját:
```
python --version
```
Ha valami hibába futnál, az [eredeti weboldalról](https://www.python.org/downloads/) tudod letölteni.
<br />
<br />
## Pygame ellenőrzése
Ezek után szükséged lesz még a Pygame-re is.
A következő paranccsal tudod ellenőrizni, szintén a paranccsorban a Pygame verzióját:
```
pip --version
```

Ha nincs feltelepítve, akkor az alábbi paranccsal tudjuk azt feltelepíteni:
```
pip install pygame
```
